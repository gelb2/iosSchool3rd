###switch문 
- 상태값은 **정수형**이여만 한다.  
- 문법도, 기능도 C언어와 같다.
- default를 안써도 된다.


  ```
    switch (상태값) {
  		case 상태값1:
  			// 해당 상태값일때 실행될 내용
  			break;
		case 상태값2:
			// 해당 상태값일때 실행될 내용
  			break;
		...
		default:
			// 해당되는 상태값이 없일때 실행될 내용
			break;
    }
  ```
